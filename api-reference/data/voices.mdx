## Example structure

```json
{
  "mist": [
    "allison",
    "alona",
    // ...and more ...
  ],
  "mistv2": [
    "abbie",
    "allison",
    // ...and more ...
  ],
}
```

## Explanation

The response object is keyed by the applicable `modelId`. This is the same `modelId` used as an argument to our [TTS APIs](https://rimelabs.mintlify.app/api-reference/endpoint/streaming-pcm).
